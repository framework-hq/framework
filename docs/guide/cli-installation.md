# CLI Installation

## Quick Start (Recommended)

Use `npx` to run the CLI without installing:

```bash
npx create-framework my-app
```

## Global Installation

If you prefer a global install:

```bash
# pnpm
pnpm add -g create-framework

# npm
npm install -g create-framework

# Then use directly
create-framework my-app
```

## Requirements

- Node.js 22 or higher
- pnpm (recommended) or npm

## Verify Installation

```bash
npx create-framework --version
```
